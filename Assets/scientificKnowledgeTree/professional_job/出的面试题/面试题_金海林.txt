1.请简述"const int & func(const int & num) const;"三个const分别的作用


答案：
第一个const：表明这个函数的返回值是一个const引用，如果函数返回的是一个引用，
             那个这个被引用的对象应该函数所属类的数据成员，加上const修饰后，
			 本类以外的对象可以使用该数据成员中的值但不能修改数据成员的值。
第二个const：引用型的传参，相对于复制型传参可以减少传参时的资源消耗，加上const
             修饰，表明可以使用参数的值，但不能修改参数的值。
第三个const：这个const是用来修饰整个函数的，表明这个函数是一个const函数，函数
             内部不能出现修改数据成员的操作，不然编译就会不通过。一般用于修饰
			 get类函数。

2.以下程序有一个错误，错误的原因是什么
#include <iostream>
struct C {
   static int s;
};
int main() {
   C c;
   C::s = 1;
}

答案：
正确程序：
#include <iostream>
struct C {
   static int s;
};
int C::s = 0;//add the line, fix error
int main() {
   C c;
   C::s = 1;
}
静态成员在类的外面需要定义，才能使用。
